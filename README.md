## 💼 프로젝트 경험

### 🍺 단국대학교 단월제 주점 자리 타이머 제작 (2024)
> **실시간 공유형 자리 관리 시스템**

🧩 **프로젝트 소개**  
단국대학교 축제(단월제) 주점 운영 효율화를 위해 **자리 이용 타이머 시스템**을 제작하였습니다.  
사용자가 자리를 배정받을 때 시작 시간을 기록하고, 시간이 만료되면 알림이 표시되어 다음 팀이 원활히 이용할 수 있도록 돕는 구조입니다.  
단순히 로컬 환경에서만 동작하는 것이 아니라, **서버와 데이터베이스 기반으로 구축되어**,  
모든 직원이 **실시간으로 동일한 자리 정보를 공유**하며, **필요 시 자리 이용 시간을 추가(연장)** 할 수 있습니다.

🔧 **기술 스택:** JavaScript, HTML, CSS, Node.js, MySQL  
🎯 **성과:** 실제 행사 운영 시 대기 시간을 줄이고 자리 회전율을 높이는 데 기여  
🧠 **역할:** 프론트엔드 UI 설계, 서버 연동 및 시간 추가 기능 구현  
🔗 **프로젝트 링크:** [👉 SW-TIMER GitHub Repository](https://github.com/helloiamhojun/SW-TIMER)
